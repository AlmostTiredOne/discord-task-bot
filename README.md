# Discord Task Manager Bot

## Setup

1. Clone

```sh
git clone git@github.com:kavin25/discord-task-bot.git
```

2. Install Dependencies

```sh
npm install
yarn
```

3. Fill .env with Discord Bot Token (Developer Portal)

```sh
cp .env.example .env
```

4. Run locally (using nodemon)

```sh
npm run dev
yarn dev
```
